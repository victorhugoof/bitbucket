# Projeto com JavaFX 11 e OpenJDK 11

### Configurações para executar:
* Configurar OpenJDK 11 no eclipse
* Configurar User Libraries JavaFX SDK11
* Adicionar no build path do projeto o JDK 11 e a SDK do JavaFX
- Run configurations -> VM Arguments
    - --module-path C:\Java\javafx-sdk-11.0.2\lib 
    - --add-modules=javafx.controls,javafx.graphics,javafx.base,javafx.fxml,javafx.media,javafx.swing,javafx.web
